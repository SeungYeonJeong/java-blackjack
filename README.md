# 연료주입

### 기능요구사항
- [ ] Sonata 2대, Avante 1대, K5 2대로 총 5대의 차량이 있다.
- [ ] 여행할 목적지의 대략적인 이동거리를 입력받는다.
- [ ] 차량 별로 필요한 연료를 주입받는다.
- [ ] 차량 별로 주입해야할 연료량을 확인할 수 있는 보고서를 생성한다.

### 프로그래밍 요구 사항
- 상속과 추상 메서드를 활용한다.
- 위 요구사항을 if/else 절을 쓰지 않고 구현해야 한다.
